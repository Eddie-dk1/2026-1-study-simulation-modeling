# Релиз лабораторной работы №2

## Общая информация

Лабораторная работа №2 посвящена literate programming и подготовке воспроизводимых материалов для моделей SIR и Лотки-Вольтерры. Релиз объединяет проект с кодом моделей, отчёт, презентацию и дополнительные методические файлы.

## Состав релиза

- `report/` — исходники и собранные версии отчёта.
- `presentation/` — исходники и собранные версии презентации.
- `lab_02_models/` — проект DrWatson с моделями и производными literate-материалами.
- `INSTRUCTIONS.md` — файл с инструкциями к лабораторной работе.
- `simulation-modeling-lab (1).pdf` — дополнительный PDF-файл, присутствующий в каталоге лабораторной работы.

## Материалы отчёта

- `report/simulation-modeling--lab02--report.qmd` — исходный Quarto-файл отчёта.
- `report/_output/simulation-modeling--lab02--report.html` — HTML-версия отчёта.
- `report/_output/simulation-modeling--lab02--report.docx` — DOCX-версия отчёта.
- `report/_output/simulation-modeling--lab02--report.pdf` — PDF-версия отчёта.
- `report/bib/` — библиография.
- `report/image/` — изображения и скриншоты для отчёта.

## Материалы презентации

- `presentation/simulation-modeling--lab02--presentation.qmd` — исходный Quarto-файл презентации.
- `presentation/_output/simulation-modeling--lab02--presentation.html` — HTML-версия презентации.
- `presentation/_output/simulation-modeling--lab02--presentation.pdf` — PDF-версия презентации.
- `presentation/speech.md` — текстовое сопровождение выступления.
- `presentation/image/` — изображения для слайдов.

## Проект моделирования

Каталог `lab_02_models/` содержит основные материалы по моделям:

- `lab_02_models/src/` — исходный код моделей.
- `lab_02_models/scripts/` — сценарии запуска и literate-скрипты.
- `lab_02_models/docs/` — производные текстовые материалы.
- `lab_02_models/test/` — тесты.
- `lab_02_models/Project.toml` — зависимости проекта.
- `lab_02_models/README.md` — инструкция по воспроизведению.

## Итог

Релиз охватывает исходный код, literate-материалы, собранные `pdf/html/docx` для отчёта и `pdf/html` для презентации, а также дополнительные файлы лабораторной работы №2.
