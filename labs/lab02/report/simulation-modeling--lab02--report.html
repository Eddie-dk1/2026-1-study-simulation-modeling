      # Теоретическая часть

  Лабораторная рассматривает две классические системы ОДУ: эпидемиологическую SIR-модель и экосистемную модель Лотки–Вольтерры.

  # Ход работы

  ## SIR-модель

Для системы S-I-R использована схема RK4. Сохранены временные ряды и график компонент. ## Лотка–Вольтерра

Для пары хищник-жертва рассчитаны колебательные траектории. Итоговые данные сохранены в CSV и используются в отчёте. ## Параметрический анализ

Ключевые метрики сведены в компактную таблицу для удобного сравнения моделей.
