            ## Repository layout

            - `labs/lab01` — Стартовый проект, организация курса, базовые вычисления и literate pipeline.
- `labs/lab02` — Непрерывные модели SIR и Лотки–Вольтерры.
- `labs/lab03` — Агентная интерпретация Daisyworld и саморегуляция системы.
- `labs/lab04` — Агентная SIR-модель с параметрическими экспериментами.
- `labs/lab05` — Аппарат сетей Петри и задача обедающих философов.
- `labs/lab06` — Реализация SIR через сеть Петри.
- `labs/lab07` — Дискретно-событийное моделирование, M/M/c и поток событий по Россу.
- `labs/lab08` — Дискретно-событийная SIR-модель.
